# serverless-crud-offers

## Running the project locally

Step 1 - clone the project

```bash
    git clone [Repo URL]
```

Step 2 - npm install

```bash
    cd projectfolder
    npm i 
```

Step 3 - Install/run dynamodb locally

```bash
    sls dynamodb install
    sls dynamodb start
```

Step 4 - run the project offline

```bash
    sls offline
```
